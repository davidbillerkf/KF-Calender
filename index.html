Update your GitHub page

This bakes all your current amounts into a new file. You then swap it on GitHub — your boss's link updates instantly.

Step 1 — Download your updated file
Click below. It saves a file called index.html to your Downloads folder with all your amounts baked in.
2
Go to your GitHub repository
Open github.com and click on your payroll calendar repository.
3
Click on index.html in the file list
You'll see your files listed. Click the one named index.html.
4
Click the pencil icon (Edit)
Top right of the file view. Then press Ctrl+A to select everything, delete it, and paste the contents of your new file. Or use the upload option below.
5
Easier: just drag and drop
In your repo, click Add file → Upload files, drag in the new index.html, scroll down and click Commit changes. Done — your boss's page updates in about 30 seconds.
Your boss's link never changes. Once you set up GitHub Pages, the URL stays the same forever. You just keep uploading new versions of index.html whenever you want to update the amounts.
